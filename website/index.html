Intro

Hey! I'm Kevin Williams, a Statistics major at Florida State University with a strong focus on cloud infrastructure, data pipelines, and automation. I'm an AWS Certified Solutions Architect with hands-on experience building containerized systems, CI/CD pipelines, and Infrastructure as Code using Terraform.

I'm interested in data engineering, cloud platforms, and scalable backend systems. Currently exploring the intersection of statistical computing and cloud infrastructure while building real-world projects that solve practical problems. Check out my recent projects to see what I've been working on.

Projects

AWS Infrastructure Automation

Built complete infrastructure as code using Terraform to deploy a production-ready web application on AWS. Includes S3 static hosting, CloudFront CDN, Route 53 DNS management, and automated SSL certificate provisioning with ACM. Infrastructure is version-controlled and fully reproducible.

CI/CD Pipeline Implementation

Designed and implemented automated deployment pipelines using GitHub Actions. Pipeline includes automated testing, security scanning, build optimization, and automatic deployment to S3 with CloudFront cache invalidation. Reduced deployment time from hours to minutes.

Container Orchestration

Deployed microservices architecture using Docker and Kubernetes on AWS EKS. Implemented auto-scaling, health checks, and rolling updates. Set up monitoring with Prometheus and Grafana for full observability of the containerized environment.
